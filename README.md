# Pop Goes the Weasel

A browser-based singleplayer game. Start at a camp with a weasel cage in the top right, a top bar with buttons (Weasel Holes, Shop, Factory). The Weasel Holes screen has nine numbered holes in a grid; weasels of various rarities pop up randomly. All logic runs in the browser.

## Features
- Camp screen with UI and navigation bar
- Weasel Holes game with 9 holes
- Weasels pop up randomly (normal, bronze, silver, three-eyed, gold, rainbow, mythic)
- No server-side code

## How to Play
1. Open index.html in your browser.
2. Use the navigation bar to switch screens.
3. Click weasels as they pop up to score points.
